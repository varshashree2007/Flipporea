# Flip The Cards Game(C)

This is a simple Flip The Cards game
developed using C programming.

##Tools Used
-TDM-GCC Complier
-Windows OS

## How to Run
1. Complie using:
2. gcc flipcards.c -o flipcards
3. 2.Run:
4. flipcards.exe

5. ## Description
6. -Grid- based card flip game
7. -Numbers are shown after tapping a card
